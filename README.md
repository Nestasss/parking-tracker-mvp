# Парковка без штрафов

MVP-сервис, который автоматически определяет момент парковки и уведомляет пользователя через Telegram.

## Функции

- Автоматическое определение парковки по GPS
- Уведомления в Telegram
- История парковок
- Поддержка Яндекс.Карт

## Структура
- backend/ # Сервер и бот
- frontend/ # PWA-приложение
- n8n-workflows/ # Автоматизация
- web-dashboard/ # Личный кабинет
- docs/ # Документация

## Установка

1. Скопируй репозиторий
2. Установи зависимости: `npm install`
3. Скопируй `.env.example` в `.env` и укажи токены
4. Запусти: `npm start`
